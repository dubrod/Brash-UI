Modal:
- `brash-modal_window` ID must be unique and match `brash-modal_btn` ID
`heroModal_obj` and `heroModal`
- Animations can be called using `data-animation="bounce"`, or leave alone to fade. Animation names must match styles in `_animations.scss`. Only put `bounce` or `fade` not `bounceIn`.

Gallery:
- Each section requires the `brash_gallery` class name
- Use `<a>` elements because Tabindex is inherited
- Attributes `data-gallery="photo"` or `data-gallery="video"` are needed to activate Carousel
